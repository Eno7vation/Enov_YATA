DB 가이드
-------------
#### 기본적으로 DB는 제공되므로, 오류가 생길시에만 아래의 프로세스를 진행해주세요.
#### 실행가이드 1번을 진행해주세요.
> ```python manage.py makemigrations accounts```

> ```python manage.py migrate accounts```

> ```python manage.py makemigrations```

> ```python manage.py migrate```
<hr/>

실행 가이드
-------------
### 1. Docker

> ```docker build -t enov_yata/django .```

> ```docker image ls```

> ```docker run -p 8000:8000 -d enov_yata/django ```

##### 1-1 어드민 회원가입

> ```docker exec -it enov_yata/django /bin/bash``` <br/>

> ```cd srv/docker-test_server``` <br/>

> ```python manage.py createsuperuser ```

##### 1-2 실행 종료 
> ```docker ps```

> ```docker stop enov_yata/django```

### 2. Anaconda
-------------
> ```conda create -n "Django_YATA" python=3.8```

> ```conda activate Django_YATA```

> ```pip install -r requirements.txt```

#### 2-1 bash

> ```python manage.py runserver 8000```

##### 2-2 어드민 회원가입

> ```python manage.py createsuperuser ```

##### 2-3 실행 종료 

> ```Ctr + C ``` or ```Ctr + Z ```
